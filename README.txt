# my-blog（GitHub Pages 極簡模板）

## 上傳步驟
1. 到你的 GitHub repo（例如 `my-blog`）
2. 點 **Add file → Upload files**，把 `index.html` 上傳到根目錄（不要放資料夾）
3. 進入 **Settings → Pages**：
   - Source: Deploy from a branch
   - Branch: main / (root)
   - Save，等待 1–5 分鐘
4. 造訪網址：`https://你的帳號.github.io/你的repo名/`

> 你的帳號：例如 `wangshiauling220-art`  
> 你的 repo 名：例如 `my-blog`，則網址為：  
> `https://wangshiauling220-art.github.io/my-blog/`
